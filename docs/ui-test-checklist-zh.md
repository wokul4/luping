# 录屏助手 UI 测试清单（中文版）

## 启动

- [ ] 窗口标题显示「ScreenRecorder 录屏助手 0.1.0-beta」
- [ ] 所有按钮和标签为中文
- [ ] 录制源下拉框默认选中 `[M0] ... 主显示器`
- [ ] 状态栏显示「状态：就绪」
- [ ] 帧率默认为 30 FPS
- [ ] 码率默认为「高 (10000 kbps)」
- [ ] 系统声音和麦克风默认勾选
- [ ] 保存路径显示正确
- [ ] 没有报错弹窗

## 录制源选择

- [ ] 显示器显示为 `[M0] 1920x1080 主显示器` 格式
- [ ] 窗口显示为 `[窗口] process.exe - 标题` 格式
- [ ] 切换录制源后信息区域同步更新
- [ ] 刷新后窗口列表正确

## 按钮状态

### 就绪状态
- [ ] 「开始录制」可用
- [ ] 「暂停」灰色不可用
- [ ] 「停止录制」灰色不可用

### 录制中
- [ ] 「开始录制」灰色不可用，文字变为「录制中...」
- [ ] 「暂停」可用
- [ ] 「停止录制」可用
- [ ] 源选择/帧率/码率/音频/路径全部禁用
- [ ] 状态栏显示「状态：正在录制」

### 暂停
- [ ] 「暂停」文字变为「继续」
- [ ] 状态栏显示「状态：已暂停」

### 继续
- [ ] 「继续」文字变回「暂停」
- [ ] 状态栏变回「状态：正在录制」

### 停止后
- [ ] 所有控件恢复可用
- [ ] 「开始录制」文字恢复
- [ ] 「暂停」恢复灰色
- [ ] 「停止录制」恢复灰色
- [ ] 状态栏显示保存的录制信息
